#### 2 异或的性质

（0）相同异或=0，不同异或=1

（1）交换率：a ^ b = b ^ a          a ^ b ^ c = a ^ c ^ b

（2）结合率：(a ^ b) ^ c = a ^ (b ^ c)                 

（3）x ^ x = 0，x ^ 0 = x

（4）a ^ b = c  --->  c ^ b = a

（5）a ^ b ^ c ^ b = a ^ c

两个数异或的结果就是两个数数位不同结果的直观表现。当区分两个数时：

num1:       			101110      110     1111
num2:      		 	111110      001     1001
num1^num2:   	010000      111     0110

n&(-n) 找到最低位的1 mask，可作为区分两个数的标志。比如mask&num1 = 0，mask&num2 != 0。
